# 华联城会员营销平台

本地生活智慧营销系统，专为县域商超打造的会员管理与营销解决方案。

## 功能特性

### 三端合一

| 角色 | 功能 |
|------|------|
| 平台管理 | 数据看板、会员管理、商户管理、活动管理 |
| 商户中心 | 经营概览、优惠券管理、顾客列表、经营报表 |
| 会员中心 | 领券中心、我的优惠券、个人中心 |

### 核心能力

- 统一会员体系（积分、等级、储值）
- 优惠券营销（满减、折扣、赠品、特价）
- 数据分析（销售趋势、核销统计、复购率）
- 多角色权限管理

## 技术栈

- **前端框架：** Vue 3 + Vite
- **UI组件库：** Element Plus
- **状态管理：** Pinia
- **路由：** Vue Router
- **图表：** ECharts

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```
